TextCatcher 隐私权限说明

发表于 2025-12-23 阅读次数：
  1. 数据收集
    本插件不收集、存储或传输任何个人数据。所有OCR识别操作均在本地进行，识别结果仅显示在用户浏览器中。

  1. 第三方服务
    本插件使用 OCR.space API 服务来识别图片中的文字。当您使用本插件时，图片数据会发送到 OCR.space 服务器进行处理。请参考 OCR.space 的隐私政策：https://ocr.space/privacy

  2. 数据存储
    本插件仅在本地存储您的API密钥（如果配置了自定义密钥），不会上传到任何服务器。

  3. 权限说明
    • activeTab: 用于访问当前标签页的图片
    • contextMenus: 用于在右键菜单中添加识别选项
    • storage: 用于存储用户设置
    • notifications: 用于显示识别完成通知
    • host_permissions: 用于获取网页上的图片
